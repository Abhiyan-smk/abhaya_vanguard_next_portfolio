# Abhaya Simkhada Vanguard Portfolio

This is the advanced React/Next.js portfolio version requested.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Motion
- GSAP-ready architecture
- Lenis smooth scrolling
- React Three Fiber / Three.js
- Custom GLSL shader plane
- Instanced particle field

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Build

```bash
npm run build
npm run start
```

## Deploy

Recommended:
- Vercel: easiest for Next.js
- Netlify: deploy as a Next.js project using `npm run build`

This is not a drag-and-drop static ZIP. It is a source-code project that needs a Node build.
